# 個人ポートフォリオサイト - 要件定義書

## 1. プロジェクト概要

### 目的
HTML/CSSの基礎を学びながら、自己紹介と実績を紹介する個人ポートフォリオサイトを作成します。

### 対象ユーザー
- 採用担当者
- クライアント
- 同業者・ネットワーキング相手

## 2. 機能要件

### 2.1 必須機能

#### ヘッダーセクション
- サイトタイトル/ロゴ
- ナビゲーションメニュー（ページ内リンク）
  - About（自己紹介）
  - Skills（スキル）
  - Projects（プロジェクト）
  - Contact（連絡先）

#### Aboutセクション
- プロフィール写真（または代替画像）
- 自己紹介文
- 経歴の概要

#### Skillsセクション
- 習得している技術スキルのリスト
- スキルレベルの視覚的表現（バーやアイコンなど）

#### Projectsセクション
- プロジェクトカード（3〜4個）
- 各カードの内容：
  - プロジェクト名
  - 概要説明
  - 使用技術
  - プロジェクト画像

#### Contactセクション
- SNSリンク（GitHub, Twitter, LinkedInなど）
- メールアドレス
- 簡単な連絡フォーム（今後の拡張用）

#### フッター
- 著作権表記
- SNSアイコンリンク

### 2.2 技術要件

#### HTML
- セマンティックHTML5タグの使用
  - `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- 適切な見出しレベル（h1〜h6）の使用
- リストタグの活用

#### CSS
- 外部CSSファイルの使用
- Flexboxを使ったレイアウト
- カラースキームの統一
- ホバーエフェクト
- 基本的なアニメーション

## 3. 非機能要件

### デザイン
- シンプルで読みやすいデザイン
- 一貫性のある配色
- 適切な余白とスペーシング

### アクセシビリティ
- alt属性の適切な使用
- コントラスト比の確保
- 論理的なタブ順序

### ブラウザ対応
- モダンブラウザ（Chrome, Firefox, Safari, Edge）での動作

## 4. 制約事項

- JavaScriptは使用しない（HTML/CSSのみ）
- レスポンシブデザインは次週以降で対応
- バックエンド機能は含まない

## 5. 成果物

- `index.html` - メインHTMLファイル
- `style.css` - スタイルシート
- `images/` - 画像ファイル用ディレクトリ
- `README.md` - プロジェクト説明書

## 6. 学習目標

このプロジェクトを通じて以下を習得します：

- HTML5のセマンティックタグの理解と使用
- CSSセレクタの基本
- Flexboxレイアウトの基礎
- カラーとタイポグラフィの基本
- ボックスモデルの理解
- 基本的なCSSアニメーション
